# WareHouseApp_Final


---

## Descripción

**WareHouseApp_Final** es una aplicación de gestión de almacén desarrollada en Java, cuyo objetivo es optimizar el control de inventario dentro de una organización.
El sistema permite administrar productos, registrar información relevante y mantener un seguimiento estructurado de los recursos disponibles.

Este proyecto fue desarrollado como parte de un proceso de formación en desarrollo de software, aplicando principios de organización, lógica de programación y manejo de datos.

---

## Objetivos del proyecto

* **Automatizar** el control de inventario
* **Reducir errores** en el manejo manual de productos
* **Mejorar la organización** de los datos
* **Facilitar el acceso** a la información del almacén

---

## Funcionalidades principales

* **Registro de productos:** permite agregar nuevos artículos al sistema
* **Consulta de inventario:** visualización de los productos almacenados
* **Búsqueda de productos:** localización rápida de artículos específicos
* **Gestión de datos:** estructura organizada para el manejo eficiente de la información

---

## Tecnologías utilizadas

* **Lenguaje de programación:** Java
* **Entorno de desarrollo:** Visual Studio Code
* **Gestión de dependencias:** Librerías incluidas en la carpeta `lib`
* **Control de versiones:** Git y GitHub
